# Roberto & Alexia Budget App

A personal budget tracker built for Roberto and Alexia.

## Deploy to Vercel (5 minutes)

### Option A — GitHub + Vercel (recommended)
1. Go to github.com and create a free account if you don't have one
2. Click "+" → "New repository" → name it `budget-app` → Create
3. Upload all these files (drag & drop the whole folder)
4. Go to vercel.com → sign up with your GitHub account
5. Click "Add New Project" → select your `budget-app` repo
6. Click Deploy — done! You'll get a URL like `budget-app.vercel.app`

### Option B — Vercel CLI
```bash
npm install -g vercel
cd budget-app
vercel
```

## Features
- 👫 Two profiles: Roberto & Alexia
- 💰 Income tracking (Job, Investments, Other)
- 🔒 Fixed expenses (Rent, Subscriptions, Utilities)  
- 〰 Variable expenses (Food, Transport, Shopping, Health, Entertainment, Personal, Travel)
- 📊 Monthly summaries & 5-month chart
- 🎯 Shared savings goals
- 📂 CSV bank statement import with AI categorization
- 🔄 Auto-sync between devices

## Add to Home Screen (PWA)
Once deployed, open the URL on your phone:
- **iPhone**: Tap Share → "Add to Home Screen"
- **Android**: Tap ⋮ → "Add to Home Screen"
